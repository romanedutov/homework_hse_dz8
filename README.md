# homework_hse_dz8
capture.py - скрипт захвата сетевого трафика
Этот файл содержит Python-скрипт для перехвата HTTP-трафика при выполнении XSS-атаки на уязвимый сайт Google Gruyere. Скрипт использует библиотеку Scapy для сниффинга сетевых пакетов. Он фильтрует трафик на порту 80, находит HTTP-запросы содержащие XSS payload и сохраняет все захваченные пакеты в файл xss_traffic.pcap. При обнаружении трафика с ID тестового экземпляра Gruyere скрипт выводит соответствующие сообщения в консоль.

analyze.py - скрипт анализа захваченного трафика
Этот файл содержит Python-скрипт для анализа ранее захваченного сетевого трафика. Скрипт загружает файл xss_traffic.pcap, читает все пакеты и ищет среди них HTTP-запросы содержащие XSS payload. При обнаружении уязвимости скрипт выводит подробную информацию о пакете: номер пакета, полный HTTP-запрос с payload, заголовки Host и User-Agent, а также общую длину данных. Это позволяет визуализировать как XSS-атака выглядит в сетевом трафике.

xss_traffic.pcap - файл захваченного сетевого трафика
Это бинарный файл в формате PCAP, содержащий все захваченные сетевые пакеты во время выполнения XSS-атаки. Файл содержит 35 пакетов, включая HTTP-запросы с XSS payload на сайт Google Gruyere. Файл может быть открыт в любом анализаторе сетевого трафика таком как Wireshark, tcpdump или Scapy для детального изучения. В пакете номер 3 содержится полный HTTP-запрос с XSS payload что подтверждает успешное выполнение атаки.
